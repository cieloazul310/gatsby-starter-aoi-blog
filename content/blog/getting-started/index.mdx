Gatsby Theme Aoi の使い方を説明します。

### インストール

Aoi Theme を利用したスターターを使う場合、

```shell
gatsby new ${your_project} https://github.com/cieloazul310/gatsby-starter-aoi-theme/
cd ${your_project}

# develop
yarn run develop

# build
yarn run build

# serve
yarn run serve

# format with prettier
yarn run format
```

もしテーマのみを個別に使いたい場合は、

```shell
yarn add gatsby-theme-aoi
```
