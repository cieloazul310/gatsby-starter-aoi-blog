### Gatsby Theme Aoi の紹介

さて、早速話が脱線しましたが、この記事は [Gatsby Theme Aoi] を紹介する趣旨で書かれています。[Gatsby Theme Aoi] と言われても読者の皆さんには何のこっちゃと受け止められるかもしれませんが、[Gatsby] と [Theme][Gatsby Theme] と Aoi、この3つの単語に分解して解説していきます。

#### Gatsby

[Gatsby] は React 製の静的ページ作成フレームワークです。

#### Gatsby Theme

[Gatsby Theme] はレイアウトやページ生成機能など特定の役割に特化させたモジュールです。
